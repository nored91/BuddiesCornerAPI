
## Description

BuddiesCorner API is made with [Nest] using [TypeORM]
Respecting https://jsonapi.org convention
Tests are handled by [Jest]

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
